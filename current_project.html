Vidshare

My main current project is a web-app for school. My assignment is to put together a web-app with users, videos, and reviews. videos and tasks are created by admins and the users must complete the review tasks assigned to them. The project is currently being built in elixir using the phoenix framework. You can see my progress Here.

Tasks are created by adding a review with 'NULL' as the comment and video review link. The review will then be displayed on the users dashboard and once it is edited be taken off.
