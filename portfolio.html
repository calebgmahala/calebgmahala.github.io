Electeams (Work in Progress)

This was the big project for our third semester. We had to build a project that pulled well structured data from a custom api using our own well structured database. The app was supposed to show the data visually(graphs), and allowed for an admin portal and data accesing. Testing was mandatory and the api needed documentation. We where allowed to use our desired languages for the project.

The project is a scheduling app designed more specifically for counter strike organizer(leagues). Users are on teams and the teams can play in tournaments or leagues run by organizations. Matches can be recorded and stats can be seen on your profile based on your matches with a specific team

This project was built with a python(Flask) api pulling from a MySQL(Google Cloud) database. The backend is built with javascript(Express) and frontend templating is done with Mustache(js). Graphs are using Chart(js)

I learned a lot from this project. Second semester was difficult for me and my class as we jumped through many languages(ruby, elixer, php) and did not get quality time with just one. This semester though, we where able to spend quality time on the languages of our choosing. I'm excited to have built my first backend project. I learned alot and even though I found lots of mistakes late in my project, I'm happy for this expierience as it has grown me as a developer
